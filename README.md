# Netflix Content Analytics Dashboard

## Project Overview
This project analyzes the Netflix Movies and TV Shows dataset using Python. It includes data cleaning, exploratory data analysis (EDA), visualizations, and business insights.

## Objectives
- Load the Netflix dataset
- Clean missing and duplicate data
- Perform Exploratory Data Analysis (EDA)
- Create visualizations
- Generate business insights

## Dataset
- Dataset: Netflix Movies and TV Shows
- File: netflix_titles.csv

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Project Structure

```
Netflix-Content-Analytics/
│── Netflix_Content_Analytics.ipynb
│── netflix_titles.csv
│── README.md
│── requirements.txt
```

## How to Run

1. Install required libraries.
2. Open the Jupyter Notebook.
3. Run all cells from top to bottom.
4. Review the generated charts and insights.

## Visualizations
- Movies vs TV Shows
- Top 10 Countries
- Top Genres
- Content Ratings
- Release Year Trend
- Top Directors
- Top Actors
- Movie Duration Distribution
- Content Added by Year

## Conclusion
This project demonstrates data cleaning, exploratory data analysis, and visualization techniques using Python to understand Netflix content trends.
